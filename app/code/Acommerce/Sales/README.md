Acommerce_Sales

### What is this repository for? ###

* Module for setting sale order 'invoiced' and 'shipped' status
* [More detail](https://bitbucket.org/acommerce/magento2-extension-acommerce-sales/)

### Module Dependency ###

* acommerce/all
